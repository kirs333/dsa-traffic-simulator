## Project Overview
A real-time traffic simulation built with Python and Tkinter that optimizes traffic flow using a Smart Queue Algorithm. Unlike traditional timers, this system dynamically adjusts green light duration based on the number of vehicles waiting in the lane.

- **Tkinter** (Standard Python GUI library).
